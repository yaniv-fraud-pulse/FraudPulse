# Gilroy Font Files

Add your Gilroy font files to this directory with the following naming:

## Required Files (WOFF2 format - recommended):
- `Gilroy-Regular.woff2`
- `Gilroy-Medium.woff2`
- `Gilroy-SemiBold.woff2`
- `Gilroy-Bold.woff2`
- `Gilroy-ExtraBold.woff2`

## Optional Files (WOFF format - fallback):
- `Gilroy-Regular.woff`
- `Gilroy-Medium.woff`
- `Gilroy-SemiBold.woff`
- `Gilroy-Bold.woff`
- `Gilroy-ExtraBold.woff`

## Where to get Gilroy:
1. If you have a license, download from your font provider
2. Convert TTF/OTF files to WOFF2 using: https://cloudconvert.com/ttf-to-woff2
3. Place all font files in this directory

## Weights:
- Regular: 400
- Medium: 500
- SemiBold: 600
- Bold: 700
- ExtraBold: 800
